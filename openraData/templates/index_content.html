OpenRA is a free and open-source RTS game engine that lets you play re-balanced games like C&C Red Alert, C&C Tiberian Dawn, Dune 2000...

The engine allows players to create new mods, add modified versions of units, buildings, maps... You can even script your own missions with Lua!


If you created a new map and want to play it with other players, you just have to:
- upload it on this website
- create a new game and select your new map
- other players will auto-download the map directly from this website, without doing anything
